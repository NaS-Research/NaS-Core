# Ingestion

Required artifacts:

- `ingestion/README.md`
- `ingestion/data_feasibility.yaml`

`data_feasibility.yaml` is a Phase 0 requirements and source-assessment document,
not an ingestion authorization. It records the minimum variables, access and
license questions, assay compatibility, independence, overlap risk, and no-go
conditions for discovery and external validation. Its outcome-data-access flag
must remain false until the applicable founder gate is recorded.

Completion gate: Governed immutable dataset snapshot is verified.
