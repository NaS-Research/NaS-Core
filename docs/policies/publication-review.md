# Publication Review Policy

No analysis or model response is automatically publishable.

A research release must identify:

- the approved source registrations and immutable dataset snapshots;
- the preregistered plan and deviations;
- code revision, environment, parameters, and executed runs;
- tables, figures, effect sizes, uncertainty, and null results;
- external evidence, contradictory evidence, and limitations;
- required dataset acknowledgements and source attribution;
- authorship, conflicts, funding, and disclosures; and
- scientific and publication approvals.

Every substantive sentence must trace to an executed artifact, an external
source, or a clearly labeled interpretation. Source publication permission and
attribution requirements are enforced before release.
